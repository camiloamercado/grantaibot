üîê Phase 5 ‚Äî Configure Secure API Key (Critical)

1. In your Netlify site dashboard, go to Site configuration ‚Üí Environment variables

3. Enter:

FieldValue

4. Click Save

5. Go back to Deploys and click Trigger deploy ‚Üí Deploy site to apply the new variable

9 Update index.html to Use Netlify Proxy

The proxy function netlify/functions/claude-proxy.js handles API calls securely. You need to update index.html to point to it instead of the Anthropic API directly.

In index.html, find this code in the callClaudeAPI function:

// FIND this line:
const response = await fetch('https://api.anthropic.com/v1/messages', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },

Replace it with:

// REPLACE with:
const response = await fetch('/.netlify/functions/claude-proxy', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },

Save the file, commit to GitHub, and Netlify will auto-redeploy.

‚úÖ After this change, your API key is fully server-side and secure.
